---
title: "Connect a model provider"
description: "Add API-key and interactive provider auth from the UI."
---

Go to Settings → Connections.

HQ supports provider auth flows exposed by OpenClaw, including API keys, OAuth paste flows, device codes, CLI reuse, and local URL flows.

The UI enqueues connection commands. The runner executes them on the gateway and streams the required state back through the command row.

After a provider is connected, set a default model globally or from an individual agent's rail.
