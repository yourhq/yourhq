---
title: "Workspace setup"
description: "Configure the workspace profile, pipeline, custom fields, and streams."
---

After onboarding, HQ redirects new workspaces to the setup wizard.

The wizard configures:

- Workspace name and slug.
- Owner profile.
- Pipeline stages.
- Contact and organization fields.
- Initial task streams.

Presets help seed common workflows such as outreach, job search, recruiting, sales, client work, networking, personal operations, or a blank workspace.

You can edit these later from Settings.
